# TRIPLE-ECHO

CSV形式の問題集を読み込んで英語学習を行うためのツールです。

## 主な機能

-   **3つの学習モード**:
    -   **通常モード**: 穴埋め、選択問題、和訳、誤文訂正などに対応。
    -   **再生モード**: 英文の自動リピート再生と自動進行。聞き流し用。
    -   **ディクテーション・モード**: 英文を隠して聴こえた音をタイピング。
-   **音声設定**:
    -   アメリカ英語 (en-US) の音声エンジンを使用。
    -   再生のたびに声の種類をランダムに切り替え可能。
    -   0.1倍速から1.0倍速まで徐々に速度を上げるスピードトレーニング機能。
-   **復習機能**:
    -   間違えた問題をブラウザに記録し、後からまとめて再挑戦可能。
-   **CSV読み込み**:
    -   独自のCSVファイルを読み込んで学習可能。

## CSVファイルの形式

以下のヘッダー（項目名）を持つCSVを読み込めます。

`"item_id","unit_category","difficulty_level","format_type","question_text","correct_answer","explanation","full_sentence","tags"`

| 項目名 | 説明 |
| :--- | :--- |
| item_id | 問題ID |
| unit_category | 単元（不定詞、接続詞など） |
| difficulty_level | 難易度レベル |
| format_type | 出題形式（穴埋め、選択問題、誤文訂正など） |
| question_text | 問題文（穴埋めは `( )` を使用） |
| correct_answer | 正解（選択肢は `/` で区切り） |
| explanation | 解説文 |
| full_sentence | 完成した英文（音声読み上げ用） |
| tags | 絞り込み用タグ |

## 使い方

1.  `index.html` をブラウザで開く。
2.  CSVファイルまたはフォルダを選択して読み込む。
3.  条件を設定して「出題開始」。
